# spotless
Lightweight Spotify client designed for the terminal. Perfect for developers and power users, it lets you control your Spotify account directly from the command line. Play music, search for tracks, manage playlists, and moreâ€”all without the need for a heavy app or GUI. Whether you're working on a coding project or simply prefer a minimalist experience, Spotless offers a fast, efficient way to enjoy your music, all from your terminal.

## Installation

### Linux / Mac
```bash
git clone https://github.com/gavrh/spotless.git && cd spotless

mkdir build && cd build

cmake .. && make
```
